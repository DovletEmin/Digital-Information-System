# Дизайн фронтенда - SMU Digital Library

## Обзор

Дизайн выполнен точно по предоставленным скриншотам с учетом всех деталей:

### Страница Makalalar (Статьи) и Dissertasiýalar (Диссертации)

**Структура:**

1. **Заголовок**: "Sizi gyzyklandyrýan temalary tapyň" (по центру)
2. **Поисковая строка**: с иконкой лупы слева
3. **Категории**: размещены в **3 колонки** горизонтально (не сайдбар!)
   - Категории расположены равномерно в 3 колонки
   - При клике на категорию она подсвечивается синим цветом
   - Повторный клик снимает фильтр
4. **Секция "Iň köp okalanlar"**: (Самые читаемые)
   - Карточки статей/диссертаций в виде списка
   - Каждая карточка содержит:
     - Категория и институция (мелким шрифтом вверху)
     - Заголовок (ссылка)
     - Краткое описание (2 строки)
     - Метаданные: дата, количество просмотров (с иконкой глаза), рейтинг (с звездочкой)
     - Изображение справа (если есть)
     - Кнопка "Oka" (Читать) - синяя круглая кнопка

### Страница Kitaplar (Книги)

**Структура:**

1. **Заголовок**: "Sizi gyzyklandyrýan temalary tapyň" (по центру)
2. **Поисковая строка**: с иконкой лупы слева и иконкой фильтров справа
3. **Кнопки типа категорий**:
   - "Okuw gollanmalary" (Учебные пособия)
   - "Basgyz kitaplar" (Печатные книги)
4. **Кнопки категорий предметов** (горизонтально, множество):
   - TDU, TMDDI, TM we TU, HYY we ÖU, TDLU, TMK, TDMI
   - TT we II, TOHU, TOHI, TITU, TDMHSI, TDEI
   - TTT we UKI, TDY we DI, TNGU
5. **Секция "Soňky goşulanlar"** (Недавно добавленные):
   - 6 обложек книг в ряд
   - Формат карточки: обложка книги с соотношением сторон 2:3
6. **Секция "Köp okalanlar"** (Популярные):
   - 6 обложек книг в ряд
   - Тот же формат карточки

## Ключевые особенности дизайна

### Цветовая схема

- **Основной цвет**: синий (primary-600) для активных элементов
- **Фон**: белый для карточек, светло-серый для страницы
- **Текст**: темно-серый для основного текста, светло-серый для второстепенного

### Типографика

- **Заголовки**: жирный, крупный шрифт
- **Категории**: средний размер, увеличивается при активации
- **Карточки**: иерархия размеров для заголовка, описания и метаданных

### Интерактивность

- Все кнопки и ссылки имеют hover эффекты
- Активные категории подсвечиваются синим цветом и жирным шрифтом
- Карточки получают тень при наведении
- Плавные transition анимации на всех элементах

### Адаптивность

- **Desktop**: полный размер с множеством колонок
- **Tablet**: уменьшение количества колонок
- **Mobile**: одна колонка для удобства чтения

## Компоненты

### ArticleListPage.vue

- Категории в 3 колонки
- Карточки статей со всеми метаданными
- Изображения справа
- Кнопка "Oka"

### DissertationListPage.vue

- Идентичен ArticleListPage
- Те же принципы дизайна

### BookListPage.vue

- Горизонтальные кнопки категорий
- Карточки книг с обложками
- Две секции: "Soňky goşulanlar" и "Köp okalanlar"
- Сетка 6 карточек в ряд

### BookCard.vue

- Специализированный компонент для обложек
- Соотношение сторон 2:3
- Hover эффекты
- Кнопка закладки поверх обложки

## Технические детали

- **Фреймворк**: Vue 3 с Composition API
- **Стили**: TailwindCSS с кастомной палитрой primary
- **Состояние**: Pinia stores для управления данными
- **Маршрутизация**: Vue Router с lazy loading
- **API**: Интеграция с Django REST API через Axios

## Запуск

```bash
cd frontend
npm install
npm run dev
```

Откройте http://localhost:3000 для просмотра.

## Важные замечания

1. **Дизайн точно соответствует скриншотам** - все элементы размещены как в оригинале
2. **Категории НЕ в сайдбаре** - они в 3 колонки горизонтально
3. **Книги с обложками** - специальные карточки с изображениями
4. **Статьи/Диссертации** - списком с текстовыми карточками
5. **Turkmen язык** - все тексты на туркменском по умолчанию
